# Megaline Statistical Data Analysis

## Overview
This project analyzes Megaline telecom customer data to understand user behavior, compare subscription plans, and identify statistically significant differences in usage.

## Objectives
- Clean and preprocess raw data
- Explore usage patterns (calls, messages, internet)
- Compare plan performance
- Test hypotheses using statistical methods

## Tools & Libraries
- Python
- pandas
- numpy
- matplotlib / seaborn
- scipy

## Project Structure
- /data — raw and processed datasets
- /notebooks — analysis notebooks
- /src — helper functions (if applicable)
